## Format

| 语法                                       | 效果                         | 快捷键         |
| :--------------------------------------- | :------------------------- | :---------- |
| \**strong**                              | **strong**                 | ⌘ + B       |
| \*emphasis*                              | *emphasis*                 | ⌘ + I       |
| \<u>underline</u>                        | <u>underline</u>           | ⌘ + U       |
| \`code`                     | `code`                     | ⌘ + shift + ` |                            |             |
| \~~strike~~                              | ~~strike~~                 |             |
| \[百度](http://www.baidu.com)              | [百度](http://www.baidu.com) | ⌘ + K       |
| ![imageName](imageSource)                |                            | ⌘ + alt + I |
| \\                                       | clear format               | ⌘ + \       |

## Paragraph

### Heading

- 语法

  - \#h1
  - \##h2
  - \###h3
  - \####h4
  - \#####h5
  - \######h6

- 效果

  - # h1

  - ## h2

  - ### h3

  - #### h4

  - ##### h5

  - ###### h6

- 快捷键

  - ⌘ + 数字键

### Code Block

- 语法

  - \~~~

- 效果

  - ~~~
    code block
    ~~~

- 快捷键

  - ⌘ + alt + C

### Quote

- 语法

  - \>引用

- 效果

  - > 引用

- 快捷键

  - ⌘ + alt +Q

### Horizontal Line

- 语法

  - \---

- 效果

  - ---

- 快捷键

  - ⌘ + shift + -

### Ordered List

- 语法
  - \1. Item
  - \2. Item 
  - \3. Item
- 效果
  - 1. Item
  - 2. Item
  - 3. Item
- 快捷键
  - ⌘ + alt + O

### Unordered List

- 语法
  - \* Item
  - \* Item
  - \* Item
- 效果
  - * Item
  - * Item
  - * Item
- 快捷键
  - ⌘ + alt + U

### Table List

- 快捷键

  - ⌘ + T

- 效果

  - | 1    | 2    | 3    |
    | ---- | ---- | ---- |
    | 4    | 5    | 6    |
    | 7    | 8    | 9    |
