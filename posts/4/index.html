include vs. import vs. class

include

C语言中引用头文件的语法，无法防止重复引用头文件。

import

OC中引用头文件的语法，可以防止重复引用头文件，无法防止循环引用头文件。

class

使用@class告知编译器有这样一个类，书写代码时不要报错，真正调用该类的方法时，再#import该类。 可以防止循环引用头文件。

一般来讲，头文件中使用@class引用其他类，在源文件中#import该类。
